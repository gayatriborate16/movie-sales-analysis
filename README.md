# 🎬 Movie Sales Analysis

## 📌 Overview
This project analyzes movie sales data to uncover trends in revenue, genre performance, and factors influencing movie success.

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Excel
- Power BI

## 📊 Objectives
- Analyze revenue trends across different genres
- Identify top-performing movies
- Study the relationship between budget and revenue
- Create interactive dashboards for better insights

## 📈 Key Insights
- High-budget movies tend to generate higher revenue
- Action and Adventure genres perform better
- Seasonal trends impact movie sales

## 📂 Project Structure
- `data/` → Dataset files
- `notebooks/` → Analysis code
- `visuals/` → Charts and graphs

## 🚀 Conclusion
This project provides valuable insights into movie performance and helps in making data-driven business decisions.
